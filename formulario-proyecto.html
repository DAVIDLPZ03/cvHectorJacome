Aplicaciones Interactivas con HTML5, CSS3 y JS

Creación de aplicaciones interactivas usando HTML5, CSS3 y JavaScript puro con validación de formularios

Objetivo del Proyecto

Diseñar y desarrollar un formulario interactivo para la página web de un currículum vitae que permita al usuario ingresar datos personales y preferencias mediante distintos tipos de controles como campos de texto, lista desplegable, radiobuttons y checkboxes, además de implementar dos efectos de interactividad con JavaScript: un mensaje emergente personalizado al enviar el formulario que confirme la recepción de los datos incluyendo el nombre completo del usuario, y un reloj dinámico que se incremente automáticamente para mejorar la experiencia visual y funcional de la página.

Herramientas Utilizadas

  • HTML5 estructura el contenido del currículum con formularios y controles variados usando etiquetas semánticas como section, form, input, textarea, select, y label.
  • CSS se utiliza mediante una hoja de estilos externa (style-cv.css) para controlar el diseño visual, incluyendo layout, tipografías, colores, tamaños y márgenes de la página.
  • JavaScript añade interactividad y validación, actualiza en tiempo real el reloj con setInterval y captura el envío del formulario para evitar recargas y mostrar un mensaje emergente con los datos ingresados.

Proceso de Diseño

  1. Planificación de un formulario para currículum vitae dividido en secciones como datos personales, educación, certificados, idiomas, experiencia laboral, competencias TIC e intereses, utilizando HTML5 y etiquetas 'section' para organizar el contenido semánticamente.
  2. Creación del formulario HTML con la etiqueta 'form' y id cvForm, incluyendo campos 'input' para datos personales, además de 'textarea', 'select', radiobuttons y checkboxes para recopilar información detallada y opciones específicas.
  3. Incorporación de un contenedor principal 'div class="container"' para organizar el contenido y facilitar el estilo, se añadió una imagen de perfil en la sección de datos personales, y se etiquetaron los inputs con 'label' para mejorar la accesibilidad y la experiencia del usuario.
  4. Incorporación de interactividad con JavaScript mediante un reloj digital que se actualiza cada segundo en un 'div' con id clock, y un manejador del evento submit del formulario que evita la recarga de la página y muestra un mensaje emergente personalizado con el nombre completo ingresado.
  5. Adición de metadatos 'meta' para definir el charset, habilitar la vista adaptable en móviles (viewport) y mejorar el SEO con descripción y palabras clave, además de enlazar una hoja de estilos externa (style-cv.css) para un diseño atractivo y responsivo.
  6. Pruebas y validaciones agregando atributos `required` en campos clave para garantizar el llenado obligatorio, y se utilizó JavaScript para validar la interacción y mejorar la experiencia con alertas personalizadas.

Resultado Final

Se desarrolló una página web con un formulario interactivo para la recopilación de datos de un currículum vitae. El diseño está organizado en secciones claras y utiliza diversos tipos de campos para captar información detallada del usuario. Se implementaron validaciones básicas para asegurar la integridad de los datos ingresados.
Mediante JavaScript, se añadió interactividad al proyecto: al enviar el formulario, se muestra un mensaje personalizado que confirma la recepción, y un reloj digital se actualiza en tiempo real para ofrecer un elemento dinámico en la página.
El proyecto combina correctamente HTML5, CSS y JavaScript, logrando una experiencia de usuario amigable, moderna y funcional, adaptable a diferentes dispositivos y pantallas.
