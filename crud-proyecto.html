Aplicación CRUD con PHP y MySQL

Desarrollo de una aplicación web CRUD con PHP y MySQL, con gestión de archivos y conexión dinámica mediante JavaScript y Fetch API.

Objetivo del Proyecto

Desarrollar un servicio web que proporcione funcionalidad a una página web mediante JavaScript, permitiendo la conexión con una base de datos para realizar operaciones CRUD. Además, el proyecto deberá incluir una funcionalidad para la subida y gestión de archivos en el servidor, así como un documento descriptivo que detalle el funcionamiento del servicio implementado.

Herramientas Utilizadas

  • PHP gestiona la lógica del servidor con archivos CRUD y protege con bind_param() contra inyecciones SQL.
  • MySQL gestiona la base de datos, almacenando ítems con nombre, descripción y archivo
  • HTML5 define la estructura de la interfaz con un formulario para CRUD y subida de archivos.
  • JavaScript gestiona la interacción del cliente y usa *fetch* para comunicarse con el backend vía API REST
  • FormData API permite enviar datos y archivos al backend desde JavaScript
  • Apache aloja los archivos .php y sirve la aplicación en rutas tipo localhost
  • El sistema de archivos del servidor almacena los archivos subidos en la carpeta "upload/"

Proceso de Desarrollo

  1. Diseño de la base de datos con tablas para almacenar ítems con campos como nombre, descripción y archivo.
  2. Configuración del backend en PHP incluyendo archivos create.php, read.php, update.php y delete.php para las operaciones CRUD.
  3. Implementación de la subida de archivos al servidor, vinculándolos con los ítems en la base de datos.
  4. Desarrollo de una interfaz simple en index.html con un formulario y una lista dinámica de ítems usando HTML y JavaScript.
  5. Integración de JavaScript con la interfaz mediante Fetch API para realizar peticiones asíncronas al backend.
  6. Pruebas y ajustes finales para verificar las operaciones CRUD, la subida de archivos y la actualización dinámica de la interfaz.

Resultado Final

La aplicación permite una gestión completa y fluida de registros con archivos adjuntos, mediante una interfaz dinámica conectada al backend con PHP y MySQL, optimizando la experiencia del usuario sin recargas.
